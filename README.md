# Api pokemon

### Requitos
- Docker

# Comandos

Para entrar no container do projeto execute:
```bash
docker compose run --rm --service-ports hyperf-skeleton
```

Instale as dependencias
```bash
composer install
```
Iniciar o projeto:
```bash
php bin/hyperf.php start
```
